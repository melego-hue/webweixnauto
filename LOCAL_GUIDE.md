# 本地Excel版本使用指南

## 设计理念

这个本地Excel版本的设计目标是：
1. **完全本地化**：无需Notion API，无需网络连接
2. **批量内容管理**：支持生成上百条内容，类似多维表格
3. **自动化标注**：发送后自动标记为"已发布"
4. **灵活扩展**：可以随时添加新博主和新内容

## 文件夹结构说明

### 文档文件夹
- `content_database.xlsx`：主内容数据库，存储所有待发布内容
- `LOCAL_GUIDE.md`：本地版本使用指南
- `README_local.md`：本地版本完整文档

### 图片文件夹
- `images/`：博主头像图片目录
  - `路飞.png`：路飞博主头像
  - `大元.png`：大元博主头像
  - `全哥.png`：全哥博主头像
  - 可以添加其他博主图片

### 脚本文件夹
- `scripts/`：所有Python脚本
  - `run_local.py`：主入口脚本（本地版本）
  - `send_from_local.py`：本地Excel发送核心模块
  - `generate_bulk_content.py`：批量内容生成脚本

## Excel数据库设计

Excel表格作为"本地多维表格"，包含以下功能：

### 字段说明

1. **ID**：自动递增的唯一标识符
2. **博主**：博主名称（用于匹配图片）
3. **内容**：朋友圈文案内容
4. **图片路径**：对应博主图片路径
5. **状态**：`未发布`/`已发布`（自动更新）
6. **创建时间**：内容创建时间
7. **发布时间**：实际发布时间（自动填写）
8. **备注**：备注信息

### 数据管理方式

1. **批量生成**：使用脚本一次性生成上百条内容
2. **手动编辑**：直接在Excel中添加/修改内容
3. **状态管理**：脚本自动更新状态字段
4. **筛选功能**：可以筛选"未发布"的内容进行发送

## 如何使用

### 第一步：初始化

```bash
# 1. 安装依赖
pip install pandas openpyxl requests pyautogui Pillow opencv-python

# 2. 准备博主图片
# 将路飞.png、大元.png、全哥.png放入images目录

# 3. 生成初始内容库
python generate_bulk_content.py --num 100
```

### 第二步：日常使用

```bash
# 发送一条内容（随机选择）
python run_local.py local

# 指定延迟发送
python run_local.py local --min-delay 5 --max-delay 15

# 模拟测试
python run_local.py local --dry-run
```

### 第三步：批量管理

```bash
# 添加50条新内容到现有Excel
python generate_bulk_content.py --add --num 50

# 添加单个自定义内容
python generate_bulk_content.py --add --blogger "路飞" --content "自定义内容"
```

## 高级功能

### 1. 定时任务自动化

可以使用Windows任务计划程序设置定时发送：

1. 每天9:00自动发送一条内容
2. 每天12:00自动发送一条内容
3. 每天18:00自动发送一条内容

### 2. 多博主内容管理

支持多个博主同时管理：
- 路飞：珠宝、易经相关内容
- 大元：心灵、情感相关内容
- 全哥：创业、商业相关内容
- 李真：艺术、美学相关内容

### 3. 内容分类筛选

可以根据博主筛选内容：
- 只发送路飞的内容
- 只发送大元的内容
- 混合发送所有博主内容

## 扩展建议

### 1. 添加新博主

1. 在Excel中添加新博主名称
2. 将博主头像放入`images/`目录
3. 在脚本中更新博主关键词映射

### 2. 自定义内容模板

可以修改`generate_bulk_content.py`中的内容模板：
```python
def generate_content_for_blogger(blogger_name):
    # 自定义不同博主的内容风格
    if blogger_name == "路飞":
        templates = ["珠宝相关内容", "易经相关内容"]
    elif blogger_name == "大元":
        templates = ["心灵相关内容", "情感相关内容"]
    # ...
```

### 3. 数据库扩展

可以扩展Excel字段：
- `分类`：内容分类标签
- `优先级`：发送优先级
- `发送次数`：统计发送次数
- `效果反馈`：点赞/评论数据

## 注意事项

1. **Excel文件权限**：确保脚本可以读写Excel文件
2. **图片文件匹配**：博主名称必须与图片文件名匹配
3. **微信窗口准备**：确保微信已启动并登录
4. **备份Excel**：定期备份Excel文件，防止数据丢失

## 故障排查

### Excel读取失败
- 检查Excel文件路径是否正确
- 检查Excel文件是否被其他程序占用
- 确保安装了`pandas`和`openpyxl`库

### 图片匹配失败
- 检查博主名称是否正确
- 检查图片文件是否存在
- 检查图片文件名是否匹配

### 状态更新失败
- 检查Excel文件是否可写入
- 检查是否有读写权限
- 检查Excel文件格式是否正确

## 性能优化

### 1. 批量处理
- 一次性生成大量内容，减少手动操作
- 脚本自动筛选未发布内容

### 2. 缓存优化
- 图片路径缓存，减少文件读取次数
- Excel数据缓存，提高读取效率

### 3. 并发发送
- 可以扩展支持多条内容连续发送
- 支持间隔时间控制

## 总结

这个本地Excel版本实现了：
1. **完全本地化**：无需云端依赖
2. **批量内容生成**：支持生成上百条内容
3. **自动化管理**：发送后自动标注状态
4. **灵活扩展**：可以随时添加新功能

适合需要批量管理朋友圈内容，且希望完全本地化运行的用户。
