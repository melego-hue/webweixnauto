## 使用方式

### 日常使用

```bash
# 进入脚本目录
cd D:\weixin\wechat-moments-auto\scripts

# 从 Notion 自动发送
python run.py notion

# 模拟测试
python run.py notion --dry-run

# 立即发送（无延迟）
python run.py notion --min-delay 0 --max-delay 0
```

### 手动发送

```bash
# 发送指定内容
python run.py send --text "朋友圈内容"

# 指定图片
python run.py send --text "内容" --image "D:\path\to\image.png"
```

### 其他功能

```bash
# 测试微信窗口
python run.py test

# 生成文案
python run.py generate

# 截取界面
python run.py capture

# 帮助信息
python run.py help
```
